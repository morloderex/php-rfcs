====== PHP RFC: Create a global login system for php.net ======
  * Version: 0.0
  * Date: 2022-05-23 
  * Author: Aaron Junker, Aaron.junker@outlook.com 
  * Status: Draft 
  * First Published at: http://wiki.php.net/rfc/global_login 

===== Introduction =====
Currently there are diffrent login systems all over php.net:

* wiki.php.net uses wiki intern login or login over main.php.net account
* pecl.php.net uses an own login system
* bugs.php.net uses own login system or login over main.php.net account
* Mailing lists are using own tracking system
* edit.php.net allows login over several platforms (also GitHub) and anonymous
* php.net user contribution notes uses no login
* php.net preferences 
* GitHub php source code uses GitHub login
